## Weavechain Java API
[https://weavechain.com](https://weavechain.com): Layer-0 For Data


### Usage

### Gradle Groovy DSL
```
implementation 'com.weavechain:api1.0'
```

### Gradle Kotlin DSL

```
implementation("com.weavechain:api:1.0")
```

#### Apache Maven

```
<dependency>
  <groupId>com.weavechain</groupId>
  <artifactId>api</artifactId>
  <version>1.0</version>
</dependency>
```
